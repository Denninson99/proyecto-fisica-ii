Modelo físico usado

Campo magnético sobre el eje de una bobina circular:

Bx = (μ0 · N · I · a²) / [2 · (x² + a²)^(3/2)]

Esta expresión aplica para una bobina circular con N espiras concentradas. En el centro de la bobina, x = 0 y el campo es máximo para los valores dados de N, I y a.
