# Proyecto de Física II

Página web para GitHub Pages con simulaciones interactivas de electromagnetismo.

- Itsamith Serrano Rojas – 2023147279

## Simulaciones cargadas

- Ley de Gauss
- Campo magnético en una bobina
- Capacitor de placas paralelas
- Propagación de ondas electromagnéticas
- Leyes de Kirchhoff y análisis de mallas

## Archivos principales

- `index.html`: menú principal con vista previa de las simulaciones.
- `gauss.html`: simulación de Ley de Gauss para una esfera aislante.
- `campo.html`: simulación de campo magnético en el eje de una bobina.
- `capacitor.html`: simulación de capacitor de placas paralelas.
- `ondas.html`: simulación de propagación de ondas electromagnéticas.
- `kirchhoff.html`: simulación de leyes de Kirchhoff.
- `style.css`: diseño general de la página.
- `js/gauss.js`: código p5.js de Gauss.
- `js/campo.js`: código p5.js de campo magnético.
- `js/capacitor.js`: código p5.js de capacitores.
- `js/ondas.js`: código p5.js de ondas electromagnéticas.
- `js/kirchhoff.js`: código p5.js de leyes de Kirchhoff.

## Publicación en GitHub Pages

1. Subir estos archivos al repositorio.
2. Ir a `Settings`.
3. Ir a `Pages`.
4. Seleccionar la rama `main` y la carpeta `/root`.
5. Guardar y esperar el enlace de GitHub Pages.


## Validación de ondas electromagnéticas

La simulación de ondas electromagnéticas fue ajustada para validar el ejemplo 32.2 del libro: diamante con f = 5.09e14 Hz, K = 5.84 y Km = 1; y ferrita con f = 90.0e6 Hz, K = 10 y Km = 1000.
