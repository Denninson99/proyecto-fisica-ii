# Proyecto de Física II

Página web para GitHub Pages con simulaciones interactivas de electromagnetismo.

## Archivos principales

- `index.html`: menú principal con vista previa de las simulaciones.
- `gauss.html`: simulación de Ley de Gauss para una esfera aislante.
- `campo.html`: simulación de campo magnético en el eje de una bobina.
- `style.css`: diseño general de la página.
- `js/gauss.js`: código p5.js de la simulación de Gauss.
- `js/campo.js`: código p5.js de la simulación de campo magnético.

## Publicación en GitHub Pages

1. Subir estos archivos al repositorio.
2. Ir a `Settings`.
3. Ir a `Pages`.
4. Seleccionar la rama `main` y la carpeta `/root`.
5. Guardar y esperar el enlace de GitHub Pages.


## Simulaciones cargadas

- Ley de Gauss
- Campo magnético en una bobina
- Capacitor de placas paralelas
- Propagación de ondas electromagnéticas
